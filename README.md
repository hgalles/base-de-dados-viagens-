# Análise de Dados - Base de Dados de Viagens

## Descrição
Projeto desenvolvido durante a graduação utilizando Python e Google Colab para analisar uma base de dados de viagens.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Objetivos
- Limpeza dos dados
- Tratamento de valores ausentes
- Análise exploratória
- Criação de gráficos para obtenção de insights

## Como executar
1. Baixe o notebook.
2. Abra no Google Colab ou Jupyter Notebook.
3. Execute todas as células.
